# Magic: The Gathering BoosterPack Simulator

## install dependencies
```Bash
pip install -r requirement.txt
```
### dependencies package
* openpyxl
* tqdm

## Usage
### openPack simulator
```Bash
python openPack.py [CardList].xlsx
```
### Caution!
[CardList].xlsx MUST be in /public directory.

### UnitTest
```Bash
python -m unittest
```
